## Recommended References

These are industry-leading examples worth studying:

### Stripe
- **URL:** https://stripe.com
- **Known for:** Clean forms, trust-building copy, subtle animations, excellent documentation
- **Best for:** Payment flows, form design, developer experience

### Linear
- **URL:** https://linear.app
- **Known for:** Keyboard-first design, information density, dark mode, speed
- **Best for:** Productivity apps, dense interfaces, command palettes

### Vercel
- **URL:** https://vercel.com
- **Known for:** Bold typography, deployment flows, gradient usage, developer tools
- **Best for:** Technical products, dashboards, onboarding

### Notion
- **URL:** https://notion.so
- **Known for:** Flexibility, blocks system, collaborative features, empty states
- **Best for:** Content-heavy apps, collaborative tools, empty state design

### Figma
- **URL:** https://figma.com
- **Known for:** Real-time collaboration, canvas interactions, component systems
- **Best for:** Creative tools, complex interactions, multiplayer features
